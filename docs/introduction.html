Cider UI is a set of UI components that bring Apple's design language to the web. Built as semantic CSS classes on top of Tailwind CSS v4, every component follows the Apple Human Interface Guidelines — from typography and color to spacing and motion.

Semantic Classes

Use .btn, .card, .dialog — clean, readable markup.

Apple HIG Typography

13px macOS Body, 17px iOS Body for prose, SF Pro type scale with Apple-standard leading.

50+ Components

Buttons, dialogs, drawers, carousels, command palettes, and more — all with dark mode.

Quick Start

Install the package and import it into your CSS:

npm install ciderui
@import "tailwindcss";
@import "ciderui";

Then use the components in your HTML:

<button class="btn">Get Started</button>
<div class="card">
  <h3>Hello, Cider UI</h3>
  <p>Beautiful, semantic UI components.</p>
</div>
