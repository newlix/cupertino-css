Variants

HTML 
<div class="cider">
  <button class="btn-primary">Primary</button>
  <button class="btn-secondary">Secondary</button>
  <button class="btn-outline">Outline</button>
  <button class="btn-ghost">Ghost</button>
  <button class="btn-destructive">Destructive</button>
  <button class="btn-primary" disabled>Disabled</button>
</div>

Default

Inside .cider, buttons are auto-styled by their type attribute — no classes needed.

HTML 
<div class="cider">
  <button type="submit">Submit</button>
  <button>Default</button>
  <button type="reset">Reset</button>
</div>

Or use classes for the same styles:

  • type="submit" → btn-primary
  • <button> (no type) → btn-secondary
  • type="reset" → btn-ghost
