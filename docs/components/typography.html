Font sizes

All typography inside .cider uses the SF Pro system font stack. Base body text is 15px with -0.015em letter-spacing.

Element Size Weight Line height
Body 15px Regular 1.47
h1 34px Bold 1.21
h2 22px Semibold 1.18
h3 20px Semibold 1.25
h4 17px Semibold 1.29
small 12px Regular —
code 13px Medium —

Headings

Headings use tight letter-spacing with automatic vertical spacing between siblings.

Body text

Paragraphs get automatic spacing between consecutive <p> elements. Links are styled with primary color and underline.

Inline text

Inline <code> gets a rounded pill with secondary background. <kbd> renders as a key cap. <small> is muted helper text. For code blocks, use the Snippet component.

Blockquote

Left-bordered italic text with muted color.

Lists

Unordered lists get disc markers, ordered lists get decimal markers. Both have vertical spacing between items.
