Cupertino is a set of beautifully crafted UI components that bring Apple's design language to the web. Built as semantic CSS classes on top of Tailwind CSS v4, it provides 27 ready-to-use components with zero JavaScript dependencies for most components.

Semantic Classes

Use .btn, .card, .input — no utility soup.

Dark Mode

Every component supports light and dark mode out of the box.

27 Components

From buttons to toasts, everything you need for a polished UI.

Quick Start

Install the package and import it into your CSS:

npm install cupertino
@import "tailwindcss";
@import "cupertino";

Then use the components in your HTML:

<button class="btn">Get Started</button>
<div class="card">
  <h3>Hello, Cupertino</h3>
  <p>Beautiful, semantic UI components.</p>
</div>
