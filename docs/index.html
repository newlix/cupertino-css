FAQ
Is it accessible?
Yes. All components follow WAI-ARIA patterns and are keyboard navigable.
Is it styled?
Yes. Semantic CSS classes with Apple HIG styling — no utility soup.
Does it support dark mode?
Yes. Every component has light and dark mode built in. Just add the .dark class.
Can I use it with any framework?
Yes. It's pure CSS, so it works with React, Vue, Svelte, HTMX, or plain HTML.
Is it customizable?
Yes. Override any CSS variable or Tailwind utility. The components are designed to be composable.
What about animations?
All interactive components include smooth Apple-style transitions that respect prefers-reduced-motion.
