Opt-in Auto-styling

.cider adds no layout styles — it purely activates element auto-styling. Add your own padding, max-width, or any layout utilities as needed.

HTML 
<div class="cider px-5">
  <h1>Hello</h1>
  <p>Auto-styled, layout is up to you.</p>
  <button>Click me</button>
</div>

Centered layout

.cider-fixed is a convenient default — centered padding with responsive max-width breakpoints. Override any value with Tailwind utilities.

HTML 
<div class="cider cider-fixed">
  <h1>Hello</h1>
  <p>Centered, responsive, and auto-styled.</p>
  <button>Click me</button>
</div>

Behind the scenes, .cider-fixed is just Tailwind utilities:

CSS 
.cider-fixed {
  @apply mx-auto px-5 max-w-xl md:max-w-2xl lg:max-w-4xl xl:max-w-6xl;
}
Breakpoint Viewport Max width
Default < 768px 576px (36rem)
md ≥ 768px 672px (42rem)
lg ≥ 1024px 896px (56rem)
xl ≥ 1280px 1152px (72rem)

Custom breakpoints

Override the built-in breakpoints with Tailwind utilities for full control — utilities always win over component-layer styles.

HTML 
<!-- Constrained fluid -->
<div class="cider max-w-4xl mx-auto">
  ...
</div>

<!-- Custom fixed breakpoints -->
<div class="cider cider-fixed max-w-6xl xl:max-w-7xl">
  ...
</div>

Reset boundary

Add .cider-reset to stop .cider styles from leaking into nested components. Useful when embedding UI components inside auto-styled content.

HTML 
<div class="cider">
  <h1>Styled by .cider</h1>
  <p>This paragraph is auto-styled.</p>

  <div class="cider-reset">
    <!-- .cider styles stop here -->
    <button>Plain button (no .cider styling)</button>
  </div>
</div>

To re-enable .cider inside a reset boundary, add .cider again — it opens a new scope:

HTML 
<div class="cider">
  <p>Auto-styled</p>

  <div class="cider-reset">
    <button>Unstyled</button>

    <div class="cider">
      <button>Styled again</button>
    </div>
  </div>
</div>

Semantic containers

.cider works on any element. Use semantic HTML tags for better accessibility and document structure.

HTML 
<body>
  <header class="cider py-6">
    <h1>My App</h1>
  </header>

  <main class="cider py-8">
    <h2>Welcome</h2>
    <p>All elements inside are styled automatically.</p>
    <form>
      <label for="name">Name</label>
      <input type="text" id="name">
      <button type="submit">Send</button>
    </form>
  </main>

  <footer class="cider py-6">
    <small>&copy; 2026 My App</small>
  </footer>
</body>
